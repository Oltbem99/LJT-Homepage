# LJT-Homepage
**Junteng Liu's Personal Academic Homepage**

This is a personal academic website for Junteng Liu, a First-year PhD candidate at HKUST NLP Group.

## Site Content

- **About Me**: Personal details, academic background, research experience, publications, awards, and contact information
- **Publications**: List of academic publications (both first author and co-author)
- **CV**: Comprehensive CV with education, research experience, publications, awards, skills, and contact

## Technologies Used

This website is built using the Academic Pages template, which is a GitHub Pages template for personal and professional portfolio-oriented websites.
